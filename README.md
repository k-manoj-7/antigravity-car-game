🚗 Anti-Gravity Car Game

A simple browser-based Anti-Gravity Car Game built using HTML, CSS, and JavaScript.

🎮 How to Play

- Use the Arrow Keys to control the car.
- Avoid obstacles and survive as long as possible.
- The score increases as you keep driving.

🛠️ Technologies Used

- HTML
- CSS
- JavaScript

📂 Project Structure

- "index.html" – Main game file
- "style.css" – Game styling
- "script.js" – Game logic

🚀 Run the Game

1. Download the project or clone the repository
2. Open "index.html" in your browser
